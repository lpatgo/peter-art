# Peter Art Site

Static HTML/CSS website for Peter Mandradjieff's art and teaching content.

## Project Structure

- Root pages: gallery, solo show, group shows, review, statement, curatorial, photography, collaborative, about.
- `teaching/`: teaching documents section pages, PDFs, and teaching image assets.
- `images/`: main site image assets.
- `styles.css`: shared site styles.

## Local Preview

Open `index.html` in a browser, or serve the folder with any static server.

## Deployment Notes

- This project is deployed as a static site.
- Keep relative paths intact when uploading subfolders.
- Verify links and image paths after each upload.
